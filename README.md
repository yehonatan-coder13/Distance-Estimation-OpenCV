# Distance Estimation using OpenCV

A Python-based image processing tool designed to detect specific environmental objects (such as doors and openings) and estimate their distance from the camera.

## Key Features
* [cite_start]**Object Detection:** Implemented using the OpenCV library[cite: 32].
* [cite_start]**Distance Estimation:** Developed an algorithm achieving 75% accuracy within a 4-meter range[cite: 25, 26].
* **Environment:** Developed and tested using Google Colab.
